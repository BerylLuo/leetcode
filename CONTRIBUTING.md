## 步骤(以jiaxin为例)
> 第一步：fork

> 第二步：拉取fork后的工程，git clone https://github.com/BerylLuo/leetcode.git

> 第三步：进入工程，git remote add upstream https://github.com/Chris0325/leetcode.git

> 第四步：同步最新代码，git pull upstream master

> 第五步：修改代码，git add, commit, push

> 第六步: 发起pull request
